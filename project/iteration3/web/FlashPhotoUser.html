Getting Started with FlashPhoto

Using brushes and filters to adjust an image

FlashPhoto is a robust program, with many different tools that are available to you. This brief tutorial will walk you through the steps necessary to:

  1. Load an image onto the canvas
  2. Use a brush to draw on the image
  3. Apply a filter to the image
  4. Save the image

After completion of this tutorial, you will possess all of the knowledge to utilize the FlashPhoto application’s features. Although we will not cover every brush or filter in this tutorial, the steps discussed will be the same for every other brush and filter.

Loading an Image onto the Canvas

Once you have successfully opened the application, it is time to load an image to the canvas. In the Controls panel, the rightmost section is labeled “Image I/O”. This is where you will load up your images.

  1. Using the file browser within the Image I/O, navigate to either a JPEG or PNG image that you want to edit.
  2. Double click on that image file name to select it.
  3. Above the “Load Image” button, you should see a line of text that reads “Will load: [your image file name]”. If this is not present, it means that you have either not selected the image file correctly, or the file is not a JPEG or PNG image file.
  4. Click the “Load Image” button.
  5. The canvas will change to selected image.

Use a Brush to Draw on the Image

You can draw on the image using any one of the brushes or tools that are included in the application. Inside the Controls panel, the leftmost column is used to select your tool and set its parameters. The “Tool Type” section can be used to select a tool. The “Tool Color” section can be used to change the color of the tool. By default, the Pen tool will be selected with Black as its color. To apply a tool to the canvas, simply press and click anywhere on the canvas and drag your cursor.

Apply a Filter to the Image

Applying a filter to the image is another simple task within the FlashPhoto application. The middle column of the Controls panel is labelled “Filters”. Nine different filters come stock within the FlashPhoto application. Some of these filters have parameters that set the intensity of the filter or other options. The best policy to become familiar with the filters is to test all of them out. If you don’t like the outcome of an applied filter, you may always use the “Undo/Redo” buttons to revert to a version of the photo before the filter was applied.

To apply the filter, simply set the parameters (if any) of the filter, and click the “Apply” button contained within the box around the filter name and parameters. The image in the canvas will be updated per your filter specifications.

Saving the Image

After you have edited the photo using brushes and filters, you will most likely want to save the image. Saving the canvas is in the rightmost column of the Controls panel.

  1. If you want to save the image with the same file name, simply click the “Save Canvas” button.
  2. If you want to save the image with a new file name, delete the image name from the “Image:” input area.
  3. Type your new file name. Make sure to append the filename with either “.jpeg” or “.png” otherwise the image will not save correctly!
  4. Click the “Save Canvas” button.

After you have completed all of these steps, you possess the knowledge necessary to begin to explore the many features that FlashPhoto has to offer. Most importantly, have fun!
