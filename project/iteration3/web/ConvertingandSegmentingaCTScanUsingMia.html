Converting and Segmenting a CT Scan Using MIA

This tutorial assumes that you have a directory of .png or .jpeg images and that you know some Linux Terminal commands.

The MIA command line interface makes it easy to apply a set of filters to multiple images at once. In this tutorial, we will apply a set of filters to a directory of .png images from a CT Scan using the Linux Ubuntu built-in Terminal software. The filters we will apply are a slight blur, followed by a quantization with 4 bins.

  1. Move your directory of images into the same directory as the MIA program. Name this directory “CTImages”.
  2. Use the Ubuntu search capability to find the “Terminal” program.
  3. Use the built in “cd” command to navigate to the directory that contains the images and MIA program.
  4. Create another folder in the directory using the mkdir command.
mkdir finishedCTImages
  1. Now is the time to run a special command that will apply the filters to all the images in your directory.
./Mia CTImages/ -blur 3.0 -quantize 4 finishedCTImages/
  • The first argument specifies that we are running the Mia application.
  • The second argument specifies the folder where all of the original images can be found.
  • The third and fifth arguments specify the filter to be applied
  • The fourth and sixth arguments specify the intensity of the filter.
  • The last argument specifies the directory where the edited images should be saved.

After running the command above, your images will be saved in the new directory.

Between the input directory and the output directory can be placed any of the following commands:

-h
-sharpen <Integer>
-edgedetect
-thresh <Float>
-quantize <Integer>
-blur <Float>
-saturate <Float>
-multrgb <Float>,<Float>,<Float>
-compare

If “-h” is found anywhere in the command line, or if there is incorrect input, a help screen will be displayed and the program will exit without editing any of the photos.
